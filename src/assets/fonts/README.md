# Custom Fonts

## Love Regular Font

Please place your "Love Regular" font files in this directory:

- Love-Regular.woff2
- Love-Regular.woff
- Love-Regular.ttf (optional fallback)

These files should be from your purchased VJ Type "Love" font family (https://vj-type.com/8-love).

Once added, the font will be automatically loaded via the @font-face declaration in `/styles/globals.css`.
